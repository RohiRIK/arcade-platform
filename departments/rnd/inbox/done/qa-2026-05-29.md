# QA → R&D: P1 Bug — Game List Shows Only 3 of 5 Games

**Priority: P1 (Blocker)**
**Date: 2026-05-29**

The CSP header in nginx.conf blocks the frontend fetch to port 3001. The frontend silently falls back to a hardcoded 3-game array missing Tetris and Space Invaders.

**Required fix:**
1. Add nginx reverse proxy: `location /api { proxy_pass http://backend:3001/api; }` in `frontend/nginx.conf`
2. Change frontend `API` variable to use relative URL: `const API = '/api'` instead of constructing `http://hostname:3001/api`
3. Update CSP `connect-src` to include `'self'` (already covered if using relative URLs)
4. Update the hardcoded fallback array to include all 5 games, or remove it entirely
