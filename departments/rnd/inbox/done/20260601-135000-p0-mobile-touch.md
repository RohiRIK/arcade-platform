## Bug 1: Space Invaders touch controls DON'T WORK AT ALL

**Root Cause:** Space Invaders (line 1048-1049) listens on `window.addEventListener('keydown')`, but touch controls (line 154) dispatch events to `document`. Events dispatched on `document` do NOT bubble up to `window`.

**Fix:** Change Space Invaders to listen on `document` instead of `window`:
```js
// CHANGE FROM:
window.addEventListener('keydown', onKeyDown);
window.addEventListener('keyup', onKeyUp);
// CHANGE TO:
document.addEventListener('keydown', onKeyDown);
document.addEventListener('keyup', onKeyUp);
```

Also update the cleanup to remove from `document`.

**Affected:** Only Space Invaders. All other games (Snake, Pong, Breakout, Tetris, Pac-Man, Frogger) already listen on `document`.

## Bug 2: Snake — first touch sets direction, dies instantly

**Root Cause:** When `waitingToStart=true`, pressing ANY arrow key both starts the game AND sets direction (line 744-745 in snake-neon-serpent.js). On mobile, users naturally press ▲ first. Snake starts moving UP from (15,10), hits top wall in ~1 second.

**Fix:** When `waitingToStart` is dismissed, ignore the direction from the first keypress. Keep default direction (right):
```js
if (waitingToStart) {
  waitingToStart = false;
  running = true;
  // Do NOT set direction from first keypress — keep default (right)
  if (!audioInited) initAmbientAudio();
  e.preventDefault();
  return;
}
```

## Bug 3: Touch controls only fire keydown, never keyup

**Root Cause:** Line 155 fires `keydown` on touchstart but never fires `keyup` on touchend. Games that track held keys (Space Invaders, Breakout, Pong) need both keydown AND keyup to work properly.

**Fix:** Add keyup dispatch on touchend/mouseup:
```js
btn.addEventListener('touchstart', e => { e.preventDefault(); fireKeyDown(); });
btn.addEventListener('touchend', e => { e.preventDefault(); fireKeyUp(); });
btn.addEventListener('mousedown', e => { e.preventDefault(); fireKeyDown(); });
btn.addEventListener('mouseup', e => { e.preventDefault(); fireKeyUp(); });
```
