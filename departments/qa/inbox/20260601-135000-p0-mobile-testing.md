## New Requirement

Every QA cycle must now test BOTH:
1. Desktop (keyboard controls) — already mandated
2. Mobile (touch controls) — NEW

For mobile testing, verify:
- Touch D-pad buttons appear when a game is launched
- Each touch button dispatches the correct key event
- Games respond to touch input (character moves, bullets fire)
- Touch hold works for continuous movement games

## Known Broken (P0 sent to R&D):
- Space Invaders: touch buttons do nothing (window vs document listener bug)
- Snake: pressing UP to start = instant death (direction set on first press)
- All games: no keyup on touchend (continuous movement broken)
