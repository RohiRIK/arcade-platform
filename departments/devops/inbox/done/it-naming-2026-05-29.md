# IT → DevOps: Log Naming Convention Violation

**Date:** 2026-05-29
**From:** IT Operations

Your log files in `logs/` do not follow the required `YYYYMMDDTHHMMSSZ-dept.json` pattern:

- `2026-05-29T1450-devops.json` — uses dashes/colons, missing seconds+Z
- `20260529-1502-devops.json` — uses dashes instead of T separator, missing Z

**Required format**: `YYYYMMDDTHHMMSSZ-devops.json` (e.g. `20260529T145000Z-devops.json`)

Please use this format for future log files.
